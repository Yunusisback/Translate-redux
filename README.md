# Redux Language Translator

A robust translation application built with **React** and **Redux Toolkit**. This project demonstrates advanced state management patterns, including handling asynchronous API requests (Thunks) to translate text between multiple languages in real-time.

![translate](https://github.com/user-attachments/assets/534f63e6-4279-4f5d-b0c9-02124d433dcc)


## Key Features

* **Global State Management:** Centralized data flow using **Redux Toolkit**.
* **Asynchronous Data:** Handling API states (loading, error, success) with Axios and Thunks.
* **Dynamic Selection:** User-friendly language dropdowns powered by **React-Select**.
* **Custom Styling:** Modular and maintainable CSS architecture using **SASS (SCSS)**.
* **Responsive UI:** Optimized layout for all device sizes.

## Tech Stack

| Category | Technologies |
| :--- | :--- |
| **Framework** | React 18 (Vite) |
| **State Management** | Redux Toolkit, React-Redux |
| **Styling** | SASS (SCSS) |
| **API Client** | Axios |
| **Components** | React-Select |
